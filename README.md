# Calculator
Basic calculator in java
**Basic Calculator – Java Mini Project**



**Project Description:**

Console-based Basic Calculator developed using Java

Performs basic arithmetic operations

User enters two numbers and selects an operator

Program runs continuously until the user exits




**Features:**

➤ Addition (+)

➤ Subtraction (-)

➤ Multiplication (\*)

➤ Division (/)

➤ Modulo (%)

➤ Continuous calculations using loop

➤ Handles division and modulo by zero

➤ Exception handling for invalid inputs

➤ Simple and user-friendly console interface



**Technologies Used:**

➤ Programming Language: Java

➤ Java Version: Java 8 or above

➤ IDE: Eclipse / IntelliJ IDEA / VS Code




**How to Run the Project:**

➤ Save the file as BasicCalculator.java

➤ Open terminal or command prompt

➤ Compile the program:

     javac BasicCalculator.java

➤ Run the program:

     java BasicCalculator



**Program Flow:**

➤ Display calculator title

➤ Take first number input

➤ Take second number input

➤ Take operator input (+ , - , \* , / , %)

➤ Perform calculation using switch-case

➤ Display result

➤ Repeat until user enters x to exit

**Sample Output:**
===== Basic Calculator =====

Enter first number: 20
Enter second number: 5
Enter operator (+, -, \*, /, %) or x to exit: /
Result: 4.0



**Error Handling:**

➤ Displays error message for division by zero

➤ Displays error message for modulo by zero

➤ Handles invalid numeric input using try-catch

➤ Shows message for invalid operators



**Concepts Used:**

➤ Java Methods

➤ Switch Case

➤ Loops

➤ Exception Handling

➤ Scanner Class

➤ Conditional Statements



**Limitations:**

➤ Console-based application only

➤ Supports basic arithmetic operations only

➤ No graphical user interface (GUI)
